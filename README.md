# Emre Can Tüzer - Cyber Security Portfolio

A modern, responsive portfolio website for a Senior Network & Security Operations Engineer. Built with Next.js and Tailwind CSS.

## Acknowledgments

This portfolio's foundation is based on an open-source template originally created by [Farhan Ansari](https://github.com/fxrhan). Huge thanks to him for providing an excellent starting point, which was then significantly customized, redesigned, and adapted to fit a professional cybersecurity context.
